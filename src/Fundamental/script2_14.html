// Example of switch statement in JavaScript. // 'switch' is used to perform different actions based on different conditions. // In this example, we check the value of 'day' and log the corresponding day of the week. // If 'day' is 1, it logs "Monday"; if 2, it logs "Tuesday"; otherwise, it logs "Other day". // The 'break' statement is used to exit the switch block once a case is matched. // The 'default' case is executed if none of the cases match. // Note: Remember to include 'break' statements to prevent fall-through. // Also, 'switch' uses strict comparison (===) for matching cases. // Here, day is set to 1, so "Monday" will be logged to the console. // You can change the value of 'day' to test different cases. //-------------------------------------------------------------------------------// Document
